judul: Optimasi Model Machine Learning Untuk Prediksi Inhibitor Korosi Berbasis Augmentasi Dataset Senyawa N-Heterocyclic Menggunakan KDE
keyword: Machine Learning, Kernel Density Estimator (KDE), Inhibitor Korosi, Augmentasi Dataset,  N-Heterocyclic.
abstrak: Penelitian ini bertujuan untuk mengoptimalkan model machine learning dalam memprediksi efektivitas inhibitor korosi dari senyawa N-Heterocyclic. Tantangan utama dalam pemodelan ini adalah keterbatasan dataset akibat tingginya biaya dan waktu yang diperlukan untuk pengumpulan data eksperimen. Untuk mengatasi permasalahan tersebut, penelitian ini menggunakan Kernel Density Estimation (KDE) sebagai teknik augmentasi data yang menghasilkan sampel virtual guna meningkatkan keragaman dataset dan kinerja prediktif model. Dataset yang dikembangkan mencakup 11 fitur kimia yang relevan, seperti HOMO, LUMO, dan Gap Energy. Model machine learning linier (MLR, Ridge, Lasso, dan ElasticNet) serta non-linier (KNR, Random Forest, Gradient Boosting, AdaBoost, dan XGBoost) dievaluasi berdasarkan Root Mean Squared Error (RMSE) dan koefisien determinasi (RÂ²). Hasil penelitian menunjukkan bahwa augmentasi data menggunakan KDE mampu meningkatkan akurasi dan stabilitas prediksi, terutama pada model non-linier seperti Random Forest dan XGBoost. Penerapan KDE terbukti efektif dalam meningkatkan performa model prediktif dan dapat direkomendasikan sebagai metode augmentasi pada penelitian serupa yang memerlukan data tambahan untuk meningkatkan ketepatan prediksi.
peneliti: Rizky Syah Gumelar, Muhamad Akrom, Gustina Alfa Trisnapradika
